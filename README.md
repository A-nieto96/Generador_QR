# Generador de Códigos QR con Flask

Este es un proyecto simple y funcional desarrollado con **Python + Flask** que permite generar códigos QR a partir de cualquier texto o URL.  
La aplicación genera el QR en tiempo real y permite descargarlo en formato **PNG**.

---

## Características

- Generación de códigos QR de forma instantánea.
- Soporta texto, números y URLs.
- El QR se descarga directamente sin almacenarse en el servidor.
- Interfaz sencilla, limpia y responsive.
- Proyecto ideal para aprendizaje con Flask.
